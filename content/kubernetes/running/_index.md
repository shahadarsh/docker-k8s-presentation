### Options for Running Kubernetes

* Managed 
    * Google Kubernetes Engine (GKE)
    * Amazon Elastic Kubernetes Service (EKS)
    * Azure Kubernetes Service (AKS)
* Self Managed
    * kubeadm
    * kops
    * KRIB
    * Kubespray
* Kubernetes for the Edge/IoT devices
    * k3s
    * KubeEdge
* Local Environment
    * Minikube
    * Docker for desktop
    * Microk8s
